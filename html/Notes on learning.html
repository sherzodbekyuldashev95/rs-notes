Notes while brushing up my mind on HTML

FreeCodeCamp's html course

  • Attributesare special words inside opening tag of an element to control its behaviour
  • When you add a lower rank heading element to the page, it's implied that you are starting a new subsection
  • Find out what does self-contained element mean?
  • Label element is used to help associate the text for an input element with and input element itself (especially for assistive technologies like screen readers)
  • Each id attribute's value must be unique from all other id values for the entire page
  • fieldset element is used to group related inputs and labels together in a web form
  • legend acts as a caption for the content in fieldset element

MDN

  • Always include attribute quotes to avoid misinterpretation by a browser
  • Metadata data about the HTML, such as the author, and important keywords that describe the document, links to css etc.
  • <meta UTF-8> specifies the character set that the document is permitted to use. utf-8 allows your page to display any language.
  • Many meta elements include name and content attributes. name specifies the type of meta element it is; what type of information it contains. content specifies the actual meta content
